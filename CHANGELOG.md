## [Unreleased]

### Added

- Key management: import, list, and remove OpenPGP public and private keys
- Block encryption for one or more recipients via slash command and context menu
- Block decryption with default key selection and passphrase support
- Vault page encryption with automatic back-links to isolated encrypted pages
- Output mode selection: replace, sibling, sub-block, or clipboard
- Optional encryption metadata output (attributes or sub-blocks mode)
- OpenPGP compatibility with GnuPG 2.x and Kleopatra (RFC 4880 / RFC 9580)
- Plugin settings UI for default key, output mode, passphrase caching, and metadata options
